# BuscaCep

## Descrição
O BuscaCep é uma aplicação feita para buscar informações de um local com base no CEP digitado.

## Tecnologias
- **HTML**
- **CSS**
- **JavaScript**
- **Font Awesome** para os ícones
- **Axios**

## Funcionalidades
- Busca o CEP por meio de uma API pública (ViaCEP)
- Retorna as informações do CEP na tela
- Validação de formulário: Verifica se o CEP digitado tem 8 digitos, se é um número e se foi digitado.

## Como usar

1. Copie o repositório
  ```bash
      git clone https://github.com/jaaocode/consumindo-api-viacep.git
  ```
2. Procure o conteúdo do repositório clonado e abra com o VS Code ou em outro editor.
3. Inicie o arquivo Index.html com a extensão Live Server do VS Code
